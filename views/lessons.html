Lesson 2 - mongodb connecting

Since my c9 account is free, I believe it doesn't let large sizes of files to be runed or because of security issues. I solved the "large files" warning by running the next command line: mongod --port 27017 --dbpath=./data --smallfiles (this command is supposing that there's already a "data" directory created).
